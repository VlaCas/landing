# Notas:
# - `Flask` es la dependencia principal utilizada por la aplicación.
# - `requests` se añade por si el backend necesita realizar llamadas HTTP a APIs externas.
# - `python-dotenv` permite cargar variables de entorno desde un archivo `.env` en desarrollo.
# - `gunicorn` es recomendado para ejecutar la app en producción (Linux). En Windows puedes usar
#   el servidor de desarrollo o herramientas alternativas.
# Si prefieres versiones distintas o quieres congelar todas las dependencias del entorno,
# ejecuta `pip freeze > requirements.txt`.
